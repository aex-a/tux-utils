# Contributing to tux-utils

tux-utils is essentially a repo for Linux utilties and scripts, if I didn't want feedback, suggestions for improvement, and contributions, it wouldn't be listed publicly, in other words, contributions are welcome, it's the only way to learn and grow effectively. 

Before submitting a PR:
- Document any needed dependencies (e.g., checking if `wlrctl` or `ss` is installed).
- Provide a brief description of rationale and intent
- Be willing to answer questions

## How to Submit
1. Fork the repo and create your branch.
2. Update the README if you are adding a new utility.
3. Submit a PR for review.
